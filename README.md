# Wiki HUD style (v2)

Изменения:
- Активная ссылка в навигации: **без бордера**, только фон.
- Верхняя плашка: слева **раздел**, рядом **подпуть** (подраздел/страница).
- Кнопки: минималистичные SVG (стрелки + солнце/луна), без цветных эмодзи/логотипов.
- Панели: более мягкие, с **слегка скруглёнными** углами.

## Run
```bash
npm i
npm run dev
```


## GitHub Pages (статическая сборка)

- Сборка статического сайта: `npm run build:pages` (результат в папке `site/`).
- Локальный просмотр: `npm run preview`.
- Деплой: workflow `.github/workflows/pages.yml` деплоит ветку `main` в GitHub Pages.

Если используешь кастомный домен (не `/repo/`), можно оставить `BASE_PATH` пустым.
