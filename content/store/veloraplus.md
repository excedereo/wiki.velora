Основные преимущества Velora+

## 1. Уникальный префикс в табе/чате
```image
src: /assets/veloraplus/tabplus.webp
align: left
width: 20%
```
```yaml
/plus
```
<br>

## 2. Уникальный градиентный никнейм
```image
src: /assets/veloraplus/gradnick.png
align: left
width: 20%
```
```yaml
/gradnick #HEX1 #HEX2
```
```callout
type: note
title: Подсказка
icon: /assets/icons/idea.png
HEX-цвет — это шестизначный код, который задаёт цвет, например #FF0000 — красный.
```
<br>
